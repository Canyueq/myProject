# 1.24
## 创建数据库
### Windows（如果已添加到 PATH）
psql -U postgres
## 或指定主机和端口（可选）
psql -h localhost -p 5432 -U postgres
### 创建数据库命令
CREATE DATABASE myapp;
## 使用navicat操作
通过建立连接来连接上本地的postgresql
密码是下载pgsql时填写密码
不填名称才能看到表结构
然后就可以创建表了
## vscode插件使用
通过ctrl+shfit+p搜索Preferences: Open Settings (JSON)
可以配置vscode插件，user就是当前用户，workspace就是当前项目，default是只读的，无法修改

# 1.23
## pnpm官方文档 https://www.pnpm.cn/
通过pnpm config set store-dir 具体路径 可以指定存储目录也就是.pnpm-store
不指定的话就会生成在磁盘根目录
### pnpm i 需要配置packages.json文件形如
```json
{
    "engines":{
        "node":">=10",
        "pnpm":">=3"
    }
}
```
### 配置启动项
```json
{
    "script":{
        "dev:frontent":"cd frontend && pnpm run dev",
        "dev:backend":"cd backend && gin main.go",
        "dev":"pnpm dev:fronted && pnpm dev:backend"
    }
}
```
